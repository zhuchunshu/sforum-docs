从低版本升级到v2.3.8之后，你需要对`user_options`表进行修改，很简单，只需要运行以下命令：

在运行该命令前，你需要cd到SForum根目录下，如果你使用docker部署，则需要进入容器终端内运行：[进入容器终端](/use/docker/operate-container.html#进入容器终端)

此命令只需运行一次！

```bash
php CodeFec plugin:userOptionsMigrate
```
